資訊社

社團特色

由一群在資訊時代中探索科技領域、領導尖端知識的建中生相互合作、共同集結而成的大型社團。

學術交流

建中資訊社為本校的大社之一，因而聚集了各種資訊領域的專才。在社團課程及活動中，互相分享每個人的知識與想法，促使大家在各個方面皆有所涉獵，激盪出創新的成果。

社團活動

建中資訊社與全台近百個社團結為友社，並主要與楓資、景資、成電，四校一起主辦活動。除了常態的聯合社課以外，還有定期的暑訓、迎新、秋遊、寒訓和春遊等大型活動。藉由活動不僅聯繫社內及友社間感情，更讓大家知道資訊社是個活潑好動的社團（絕對不是油宅）！

未來領航

建中資訊社在全台灣至全世界佈下資訊科技的人才，我們不只侷限於僅僅一個高中的社團，更放眼至全世界最頂尖的資訊社群。

frontend develop

想知道網頁是怎麼寫出來的嗎？這個課程會教你如何寫出一個網頁，其中會學到HTML5/CSS和JS。從基本語法一直教到進階的相關應用。學完這個課後相信大家都有能力學會如何製作網站

Python

Python是全世界最實用、最好入門的程式語言！讓我們來看看這條蟒蛇有什麼神奇妙用吧 XD
1.安裝與環境 -- 工欲善其事，必先利其器。
2.基礎語法 -- 循序、選擇、迴圈、函數、物件、一大堆東西。
3.模組與套件 -- 前人種樹，後人砍樹。
4.數學運算 -- NumPy，一個強大的數學運算工具。
5.物理模擬 -- VPython，取代物理引擎的簡易方式。
6.網路爬蟲 -- requests，學會發送HTTP請求！

C++

課程中除了會讓大家熟悉C++的指令、語法、STL（標準函式庫），更會帶大家了解演算法到底是什麼樣的東西以及怎麼樣是好的演算法。什麼是演算法呢? 其實它大概像是一個食譜或SOP，我們把清楚的步驟用程式碼讓電腦執行多到無法想像的基本運算，為了分析日常生活中的問題及增快執行的效率，針對不同特定問題的演算法已經被大量發明，我們在課程中會依照同學學習狀況和大家逐步分享。

Machine learning

人工智慧、機器學習是現在當紅的話題，alpha GO、影音辨識、自動駕駛等等都可見到它的蹤跡。從最基本的觀念開始，了解類神經網路背後的理論及數學原理，進一步實作自己的人工智慧，甚至學會如何應用在更廣泛未知的領域！

硬體

資訊並不只有1與0的傳輸及改變，你是否希望對承載這一切的機器有更進一步的了解呢，你是否會希望能判斷為何一台筆店多了一顆水果就可以售價高昂，抑或是你是否希望能夠知道為何別人的電腦能比自己的更順暢，還是你只是想知道如何省檢測費自行DEBUG電腦的問題以及如何自己維修簡單的硬體故障，這堂課會一步步地帶著你從硬體蒙新變成能理解為何電腦這麼貴、和便宜。
